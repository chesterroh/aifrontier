## 开场    *00:00*

## 崔升准 = 媒体艺术家 + 韩美幼儿园创办人    *00:06*

<span class="paragraph-timestamp" data-ts="00:06">00:06</span> **卢正锡** 今天我们的一位联合主持人升准，到底是做什么工作的，我想就从这个话题开始。第一次见到升准是在 2008 年。现在已经快 20 年了。当时我认识的升准，官方头衔是媒体艺术家。他会用 Microsoft XNA 之类的工具做出各种很有趣的东西给我看。

我后来发现他大学主修是物理，但他还有另一个身份。升准有很强的工程背景，同时作为媒体艺术家，他也在国民大学讲课，还给多个软件项目授课，也非常活跃地进行写作。他的兴趣看起来非常广，但你会发现最后总是回到教育。他经常谈这个。

这也成为我们在 AI 时代去追问教育必须如何改变的一个契机。这是我们开这个播客的重要目的之一。在这个意义上，升准贡献很大。所以今天是升准特辑。我们会聊聊为什么升准全力投入教育、AI 和教育之间有什么关联、以及升准具体在做什么。今天就深入这些主题。

<span class="paragraph-timestamp" data-ts="01:47">01:47</span> **崔升准** 既然你问到了，我就讲讲。我自己很喜欢追随有趣的好奇心去学习，也一直在做和教育相关的工作。在大学里，我长期给设计或艺术专业的学生上 coding 课。所以在 2010 年代我在这方面很活跃。

我也在运营一所幼儿园。不过把我在幼儿园学到的方法应用到大学教育里，怎么说呢，是一段很有意思的时期。

## 为什么教育如此困难且充满争议？    *02:25*

<span class="paragraph-timestamp" data-ts="02:25">02:25</span> **崔升准** 所以今天这些故事可能会自然地讲出来。也就是我对学习和教育的想法。首先，我用 William Gibson 的话“未来已来，只是分布不均”，把它作为标题。

我很喜欢 Kieran Egan 的那篇《Why is Education So Difficult and Contentious?》。读完后我有点释然，原来不是只有我这样想，也有学者认为教育内在就有某种不可能性。越看教育，越觉得困难。

这里有一张图，卢梭的发展观、柏拉图的学术理想、社会化，这些都被放进学校里，而且因为都看起来是好东西，我们就说要把它们都做好、做出协同。但在现实现场里，这些有时会互相牵制。所以我不展开太深，但我对教育内在的不可能性非常有共鸣。

## AI 时代的岗位冲击如何影响教育    *03:31*

<span class="paragraph-timestamp" data-ts="03:31">03:31</span> **崔升准** 这段时间很多人都对教育感兴趣，也说教育要改变，但这其实是个很难谈的话题。

但从我的角度看，教育在这个方向上很难靠自己改变。我前面提到不可能性，再加上现在的现实……很多人都在说，也确实有一种挫败感。

如果岗位被冲击了，我们现在在 Fugitives' Alliance 讨论 AI 时也一直在讲这些问题。岗位被冲击的话，会发生很多令人难过的事。

<span class="paragraph-timestamp" data-ts="04:06">04:06</span> **卢正锡** 你说得比较委婉，但岗位被冲击，意思就是大裁员时代要来了。可以这么理解吧？

<span class="paragraph-timestamp" data-ts="04:15">04:15</span> **崔升准** 可以这么看，总之我觉得会有很多难过的事发生。一个我们还没经历过的世界正在到来。那教育也必然会被冲击。如果岗位被冲击。我在这里用了“再奠基（re-foundation）”这个表达，有一种重新打地基的感觉。因为在摇晃。会不会这反而是一次重新获得机会的时机？我是这么想的。也就是为既有教育的崩塌做对冲，当然不一定会崩塌，但如果崩塌了，我们在对冲的同时可以探索新机会。也许时机正好。那样的话，反对预言式教育学的力量可能会重新变强。

## 对预言式教育学的批判    *05:03*

<span class="paragraph-timestamp" data-ts="05:03">05:03</span> **崔升准** 不过这里“预言式教育学”这个词可能有点难。

它是指把已经结构化好的东西灌输并传递知识。不是说它不好，但如果过度偏向这一边，就成了预言式教育学。

也就是认为只要把它 program 化、做出 manual、教给人，人就会做好。甚至把高阶学习也拉进来这样做的那一边。

<span class="paragraph-timestamp" data-ts="05:29">05:29</span> **卢正锡** 在韩国常说“填鸭式教育”。填鸭式教育和预言式教育有关吗？

<span class="paragraph-timestamp" data-ts="05:37">05:37</span> **崔升准** 有关系。就是把东西放到轨道上让它笔直前进的方式。大概就是这种方法。

<span class="paragraph-timestamp" data-ts="05:45">05:45</span> **卢正锡** 用我们的话来说，像 supervised learning 吗？

<span class="paragraph-timestamp" data-ts="05:50">05:50</span> **崔升准** 其前提是“拥有大量知识”这件事，我认为其实已经被冲击了。我觉得它应该被冲击，而且这个过程已经开始了。所以我试着换个角度解读：我们为什么一定要学得那么快？为什么总要去学那些看起来总有标准答案的东西？我一直很好奇。

明明可以慢慢地、愉快地学，为什么必须被不确定的未来追着跑去学？为什么必须牺牲今天令人满足的学习？为什么必须渐渐失去动机？我会想到这些问题。

但如果结果反正会变成这样，“这样”会在下一章解释，会变成什么样。凭一种直觉，如果不只是 OpenAI，而是所有东西都这样被接管，没有中间地带，大家都被挤出去，如果反正会这样。那我们还要复制我们认为当下最优的 best practice 吗？我开始怀疑。我们正在进入一个谁也不知道会如何展开的不确定地带。还有必要继续复制我们现在相信的最佳实践教育吗？

<span class="paragraph-timestamp" data-ts="07:05">07:05</span> **卢正锡** 在学校成绩单拿 1 等级、修能考高分、进首尔顶尖大学好专业，这算 best practice 吗？

<span class="paragraph-timestamp" data-ts="07:15">07:15</span> **崔升准** 对，某种意义上是。那样还会有效吗？

<span class="paragraph-timestamp" data-ts="07:19">07:19</span> **卢正锡** 现在我也不知道。其实现在不只是学生，职场人、后辈前辈，大家都在经历某种心理性失范。我觉得我们才刚开始。反应快的人，很多订阅者也是这种前瞻型的人，他们已经开始想“哇，世界要被彻底翻过来了”。他们已经有了这种问题意识和危机感，但说实话我们也还没有答案。谁都还没有答案。但“旧方式不会继续了”这种感觉……

<span class="paragraph-timestamp" data-ts="07:59">07:59</span> **崔升准** 有这种感觉。

<span class="paragraph-timestamp" data-ts="08:00">08:00</span> **卢正锡** 这感觉好像已经普遍化了。是的。

<span class="paragraph-timestamp" data-ts="08:03">08:03</span> **崔升准** 所以我提出了这个问题。也许应该恢复一种更投资“今天满足感”的教育，也许有机会。

因为我们不知道未来，那与其一直对冲、焦虑未来、把东西不断往前拉、用那种线性方式学习，不如多投资一点今天的满足感。反正不知道未来会怎样，今天先满足，反而可能损失更少。

这是我想到的想法。因为这也许是可能的。

## 所有答案都在幼儿园里：Reggio Emilia 方法    *08:38*

<span class="paragraph-timestamp" data-ts="08:38">08:38</span> **崔升准** 所以现在，我是借着“我真正需要知道的一切都在幼儿园学到”这句话来搭建逻辑的，我记得那好像是很早以前的电影。

<span class="paragraph-timestamp" data-ts="08:46">08:46</span> **卢正锡** 有的。有一部很有名的电影或者书。

<span class="paragraph-timestamp" data-ts="08:50">08:50</span> **崔升准** 更准确说，是从幼儿教育里，我搭建了“也许有可学之处”的论证。就是这么搭起来的。

所以我们得到很多启发的地方，是意大利一个叫 Reggio Emilia 的小镇。去那边研修的话，不知道现在怎样，以前他们总会展示这张图。

## 10 个月大的婴儿与时钟：孩子会自己形成假设    *09:11*

<span class="paragraph-timestamp" data-ts="09:11">09:11</span> **崔升准** 这是 1981 年拍的图。是一个 10 个月大的婴儿。现在看一共四张照片，我先给你看其中三张。一个孩子，不，甚至还不能叫孩子，是婴儿。婴儿用手指着杂志上的手表，老师刚好戴着机械表，就这样拿给婴儿看。然后让婴儿听声音。

婴儿不会说话，但通过表情在交流，于是出现了一个好奇的情境。并且发生了一次经验。接下来是：从杂志上的手表图像出发，婴儿在想它会不会发出声音。婴儿虽然只有 10 个月，但如果从胚胎期算起，也大约是 20 个月的生命体。还不会说话，但在说出语言之前，已经可以形成假设并实验。

而把这个瞬间记录下来，这个强有力的画面、强有力的意义，讲述这个故事的地方，就是意大利小镇的一种教育方法。其实那里现在也不算小镇了，世界各地的人都会去看这种教育方法。那里会讲很有名的“The 100 Languages of Children”的故事，我们也受其启发，讲类似的故事。

## 不是 program，而是 project：不可预测的学习过程    *10:39*

<span class="paragraph-timestamp" data-ts="10:39">10:39</span> **崔升准** 当别人问我们做什么教育时，我们会说是 programmazione 的反面。programmazione 在意大利语里就是“program”。意思是我们实践的是 program 的反面教育。通常你说在做教育，别人会问“有什么 program？”“program 是什么？”对吧？“program”这个词里，“执行既定计划”的语感更强。但所谓不是 program，意思是在教育里的一些初始假设，真正教起来会和现实不同，并且会边做边变。因为对象是人。

所以学习并不是一种像 program 那样按可预测、渐进步骤推进的一维问题。知识形成是群体共同经历的过程。而孩子就像前面那张图里一样，会自己创造理论。如果提前把答案给他，剥夺了他生成理论的机会，其实就是拿走了它。

这也许会重复过去人的历史，但本来是我自己也能摸索出来并尝试的。提前给答案就是 spoiler。有一种哲学是：不要过度泄露那些本可被自己发现的东西。应该支持他们自己发现。这种教育大概就是这样的图景。

## 教师的角色：迷宫中的阿里阿德涅之线    *12:10*

<span class="paragraph-timestamp" data-ts="12:10">12:10</span> **崔升准** 所以我们也受其启发，尝试用自己的方式做图示。在不确定情境里，孩子和老师一起对话、探索、试错，然后想到“这个游戏也许能发展成什么”。我们把它叫做“游戏种子（play seed）”。

即便在这种健康期待的情境里，也不是直线。图上画的是曲线。意思是不会一下子直线前进。在这个点上，我们老师会说“要守护‘思量时间’”。思考的时间，琢磨的时间。不是快速地这个教完接着教那个，而是有点迂回游走的感觉。

但那里有一个点。可能是听孩子讲故事的时间，也可能是老师们共享记录并交换反馈的时间，也可能是与家长分享这些内容的场域。但那个瞬间是非常关键的点。因为那是探索潜在方向并转向新方向的机会。

所以我们录 RL 那期时，成贤你讲过 high-entropy tokens。当路径可以有很多条时，在这种不确定情境里会非常关键。从那里走哪条路，推理路径有机会变得更好。那一刻做什么，会让生成质量发生巨大变化。你获得一个 latent state。

所以我觉得这是非常相似的心理图像。意思是可以这样持续展开、持续前进。也许会在不同支流再次相遇，也可能并行推进。所以在教育领域，这些年经常把它比作意面。不只是 Reggio Emilia，教育学里常讨论这种像意面一样纠缠、没有起点终点、不断分叉出新方向的感觉。

学习是一个建构意义的过程。希腊罗马神话里有忒修斯的迷宫，对吧？米诺陶的迷宫。但忒修斯能走出迷宫，是因为一位叫阿里阿德涅的公主给了他一团线。给了线之后，在迷宫里就会留下他走过路径的记录。正因如此他才能走出来。

所以他们认为老师可以扮演这个角色。老师是拿着线团的人，通过编织关系网络，让互动与沟通成为重要经验。所以这套方法很强调教师的能力与角色。刚才也用了“像迷宫一样纠缠”的隐喻。

## 追踪学习足迹时，文档记录的重要性    *15:14*

<span class="paragraph-timestamp" data-ts="15:14">15:14</span> **崔升准** 对一场勇敢的航行来说，留下轨迹某种程度上很重要。你会知道方向，知道该往哪走，知道哪里是安全的，也能知道做过哪些实验和尝试，数据积累其实很重要。把这一点强调到教育里，就是前面说的 Reggio Emilia 在做的事。

我们也受其启发，从 1997 年起就把 documentation 工作看得很重要。所以我现在能比较有把握地引用“未来已来，只是分布不均”，因为我要介绍我们自己的案例。这是我参与过的。

1997 年我们尝试了什么呢，从我们在意大利学到东西开始，我们把所有记录都数字化。简单给大家看，这是最早的一份记录。我们 1984 年开始办园，但从 1997 年开始这样记录。我们和孩子讲过并共享的故事、他们画的图、我们做过的试错等等，第一次被这样完整记录下来。但这仍然是可阅读记录。

我们做这些事时，2000 年代初，有位在瑞典推动教育改革的人叫 Harold Göthson，他来我们机构时问：“我看你们记录做得很好，但谁在读这些？”当时我有点震惊，回答也很无力。我们只是勤于记录，但在“如何被消费”的结构上比较薄弱。

## 从 Trello 到 LLM：教育记录方式的演进    *17:15*


<span class="paragraph-timestamp" data-ts="17:15">17:15</span> **崔升准** 到了 2010 年代，2008 年、也就是 2010 年代之前，我们就引入了 Google Docs 之类的工具让组织内部共享记录。为了更主动地做这件事，大概 2013 年受 Kanban 系统启发，是 Joel Spolsky 吗？他做过一个叫 Trello 的工具。我们从 2013 年开始使用这种“像卡片一样移动并留下记录”的工具，持续记录游戏或通过游戏进行学习的过程。

但现在我觉得这和当下时代不太贴合。它很好，开发者也知道，Atlassian 收购后很多产品会变差。Trello 被 Atlassian 收购后也有些不尽如人意，所以我们从去年开始在逐步脱离 Trello。

作为把课堂发生的事情整体看清的工具，它非常优秀。但在 LLM 时代，我觉得应该把这些直接喂给 AI。所以我们今年年初做的是可以直接输入 Google Docs 的东西。一个简单编辑器，拖图进去会立刻按合适尺寸调整，粘贴进 Google Docs 后能生成格式化版本，我们用 vibe coding 做了这样一个工具。然后……Google Docs 大概今年年初也开始支持 tabs。可能有人还不知道 tabs，但现在可以这样持续写。可以分成多个类别做记录。老师们就用之前那个工具，在 Google Docs 里记录某些教育活动如何推进等内容。

但实际上即便有 tabs，Google Docs 仍然是序列化状态。然后不同游戏活动有多条记录，有些互相缠绕，也有独立不缠绕的流。你可以把这些都放到一个 context 里直接给 AI。如果只是把 Google Doc 丢给它。之所以这么容易，是因为在 NotebookLM 里把 Google Doc 当 source 给进去就能同步。文档变化，NotebookLM 也会跟着变化，老师就能立刻获得 insight。

我们也做了让这件事可行的实验。所以这是我们现在正在尝试的。我们正在做适配 LLM 时代的尝试。

我在这里想强调的是，在这种不确定情境下尝试新东西时，留下记录会有帮助。但你也能猜到，这需要很多工作。因为要记录，就要拍照、整理，还要花时间写下思考。持续的两难是：我知道“这样做是好的”，但一直在想怎么把它做得更简单。前面讲到的阶段仍需要老师亲自记录。记录时会思考，这当然有好处，但也非常耗费精力。先把这一点指出来。

## 创造新职业：Atelierista 与 Pedagogista    *38:02*

是什么呢？他们创造了岗位。教师以外的岗位。有两个岗位：一个叫“atelierista”，另一个叫“pedagogista”。Atelierista 是每个机构配一个，不是老师，而是专业艺术家。我不是 atelierista，但每个机构都有一个类似我这样的人，只是形式不同。能创造出这样的岗位差异其实非常大。引入外部视角并做融合并不容易，但他们做了。Pedagogista 作为教育专家，不是每个机构都有，而是每 3-4 个机构配一个。和我们国家的学监概念不太一样。总之这是一个重要点。

总之，他们建立了假设，孩子和老师一起建假设，并形成一种以“活实验室”去验证的文化。但问题是，这件事在 scale up 上会遇到困难。
