## チェ・スンジュン = メディアアーティスト + ハンミ幼稚園創設者    *00:06*

<span class="paragraph-timestamp" data-ts="00:06">00:06</span> **ノ・ジョンソク** 今日は共同ホストの一人であるスンジュンが、実際に何を仕事にしているのか、その話から始めようと思います。僕がスンジュンに初めて会ったのは2008年でした。もうほぼ20年前ですね。当時会ったとき、スンジュンの肩書きはメディアアーティストでした。Microsoft XNAのようなものを使って、いろんな面白い作品を作って見せてくれました。

専攻は物理学だと知ったんですが、ほかにも肩書きがありました。スンジュンには強いエンジニアリングのバックグラウンドがあって、メディアアーティストでありながら国民大学で講義もして、いくつかのソフトウェア関連プログラムでも講義し、執筆活動も活発でした。関心がすごく広く見えるのに、見ていると結局いつも教育に戻ってくるんです。教育の話を本当によくします。

それが、AIの時代に教育はどう変わるべきかを僕たちが問うきっかけにもなりました。それがこのポッドキャストを始めた大きな目的の一つでした。そういう意味でスンジュンは大きく貢献してくれています。なので今日はスンジュン特集です。なぜスンジュンが教育にオールインしているのか、AIと教育はどうつながるのか、そして具体的に何をしているのか。今日はそこを掘り下げます。

<span class="paragraph-timestamp" data-ts="01:47">01:47</span> **チェ・スンジュン** 聞かれたので話しますね。僕自身、面白い好奇心を追って学ぶのが好きで、教育に関わる仕事もしてきました。大学ではデザインやアートを専攻する学生向けに、長い間コーディングの授業をしていました。2010年代はその領域でかなり活動していました。

それから幼稚園も運営しています。幼稚園で学んだ方法を大学教育に適用するのは、何というか、面白い時間でした。

## なぜ教育はこれほど難しく、論争的なのか？    *02:25*

<span class="paragraph-timestamp" data-ts="02:25">02:25</span> **チェ・スンジュン** なので今日は、そういう話が暗黙的に出てくると思います。僕の学びと教育に関する考えですね。まず、タイトルにはウィリアム・ギブソンの言葉を引用して「未来はすでにここにある。ただ均等に分配されていないだけだ」と付けました。

Kieran Eganの「Why is Education So Difficult and Contentious?」をとても興味深く読みました。読んでみて、教育には本質的な不可能性があると考える学者がいることに、少し救われた気がしたんです。自分だけがそう思っていたわけじゃなかった。教育は見れば見るほど難しい。

ここに図がありますが、ルソー的発達、プラトン的な学問的イデア、社会化など、こうしたものが学校に全部集められています。どれも良いものだから、うまくやろう、シナジーを出そう、と言いますよね。でも現場では、それらが互いを妨げることもある。深くは話しませんが、教育に内在する不可能性に共感しました。

## AI時代の雇用崩壊が教育に与える影響    *03:31*

<span class="paragraph-timestamp" data-ts="03:31">03:31</span> **チェ・スンジュン** 最近は教育に関心を持つ人が多くて、教育を変えたいとも言いますが、実際にはとても難しいテーマです。

でも教育は、僕の見方では、この方向では自力では変わりにくいと思います。さっき不可能性の話をしましたが、最近の現実もあって……多くの人が言うように、少し敗北感のようなものがあります。

雇用が崩壊するなら、僕たちは今Fugitives' AllianceでAIの話をしながらずっとこういう問題を扱っていますが、雇用が崩壊したら悲しいことが起きます。

<span class="paragraph-timestamp" data-ts="04:06">04:06</span> **ノ・ジョンソク** かなり遠回しな言い方だけど、雇用が崩壊するってことは大量解雇の時代が来る、そう読むべきってことだよね？

<span class="paragraph-timestamp" data-ts="04:15">04:15</span> **チェ・スンジュン** そう見ることもできますし、とにかく悲しいことがたくさん起きると思います。まだ経験したことのない世界が近づいています。そうなると教育も当然揺さぶられます。雇用が崩壊するなら。だからここで僕は「再基礎化」という表現を使いました。基礎石をもう一度据え直す感じです。土台が揺れているので。これを、もう一度チャンスを得る機会にできないか、と考えています。つまり既存教育の崩壊にヘッジしながら、崩壊しないかもしれないけれど、崩壊した場合に備えて新しい機会を探る。今はそのタイミングかもしれない。そのとき、予言的教育学に反対する力が再び力を取り戻せるのではないか、と思っています。

## 予言的教育学への批判    *05:03*

<span class="paragraph-timestamp" data-ts="05:03">05:03</span> **チェ・スンジュン** ただ、ここで言う「予言的教育学」という言葉は少し難しいかもしれません。

すでに構造化されたものを注入して知識を伝達する話です。悪いということではないですが、その一方向に過度に傾く側面が予言的教育学です。

プログラムを作って、マニュアルを作って、教えればうまくいくと信じる側。そして先行学習まで引き込んで進める側です。

<span class="paragraph-timestamp" data-ts="05:29">05:29</span> **ノ・ジョンソク** 韓国では「詰め込み教育」という言い方が一般的だけど、詰め込み教育と予言的教育は関係ある？

<span class="paragraph-timestamp" data-ts="05:37">05:37</span> **チェ・スンジュン** 関係あります。何かをレールに載せて一直線に進ませるアプローチです。そういう性格ですね。

<span class="paragraph-timestamp" data-ts="05:45">05:45</span> **ノ・ジョンソク** 僕たちの言葉で言うと、supervised learningかな？

<span class="paragraph-timestamp" data-ts="05:50">05:50</span> **チェ・スンジュン** その前提として、知識をたくさん持つこと自体が、もう崩れ始めていると僕は思っています。崩れるべきだし、すでに始まっていると思う。だから僕はそれを少し別の角度で解釈しようとしました。なぜ僕たちはこんなに急いで学ばなければならないのか。なぜいつも正解があるようなものばかり学ばなければならないのか。ずっと気になっていました。

ゆっくり楽しく学べるのに、なぜ不確かな未来に追われて急いで学ばなければならないのか。なぜ今日満たされる学びを犠牲にしなければならないのか。なぜ動機を少しずつ失っていかなければならないのか。そういうことを考えてしまいます。

でも、どうせこうなるなら。「こうなる」が何かは次の章で説明します。直感的に、OpenAIだけでなく全部がこうやって奪われていくなら、中間地帯なく、みんな押し出されるなら、どうせそうなるなら。いま最善だと信じる実践を再生産するべきなのか、と考え始めます。誰にも展開が読めない不確実な領域に入っていくのに、今の最善実践と信じる教育を再生産する必要があるのか。

<span class="paragraph-timestamp" data-ts="07:05">07:05</span> **ノ・ジョンソク** 通知表でオール1を取り、CSATで高得点を取り、ソウルのトップ大学の良い学科に入る。それを最善実践と見なすべきか、ってことだね？

<span class="paragraph-timestamp" data-ts="07:15">07:15</span> **チェ・スンジュン** そう、ある意味で。果たしてそれで通用するのか。

<span class="paragraph-timestamp" data-ts="07:19">07:19</span> **ノ・ジョンソク** 今はわからない。実際いまは学生だけでなく会社員も後輩も先輩も、みんなある種の心理的アノミーを経験していると思う。まだ始まりの段階だけど。早く気づいた人たち、うちの視聴者にもそういう先見的な人が多いけど、そういう人たちは「世界が完全にひっくり返るぞ」と感じ始めている。問題意識や危機感は持ち始めたけど、正直、僕たちも答えを持っていない。まだ誰も答えを持っていない。でも昔のやり方が続かないという感覚は……

<span class="paragraph-timestamp" data-ts="07:59">07:59</span> **チェ・スンジュン** その感覚はありますね。

<span class="paragraph-timestamp" data-ts="08:00">08:00</span> **ノ・ジョンソク** 広く共有されるようになった感じがある。うん。

<span class="paragraph-timestamp" data-ts="08:03">08:03</span> **チェ・スンジュン** だからその問いを投げました。つまり、今日の満足にもっと投資する教育を回復する必要があるかもしれないし、チャンスかもしれない。

わからないからです。なら、ヘッジして未来を心配して、常に前倒しで直線的に勉強するより、今日の満足にもう少し投資する。未来がどうなるかわからないなら、今日満足することが損失を減らす道かもしれない。

そんな考えに至りました。可能かもしれないからです。

そして、僕が、僕たちが多くのインスピレーションを得た場所が、イタリアの小都市Reggio Emiliaです。そこへ研修に行くと、最近はわかりませんが、昔はいつもこの写真を見せてくれました。

## 生後10か月の子どもと時計: 子どもは自分で仮説を立てる    *09:11*

<span class="paragraph-timestamp" data-ts="09:11">09:11</span> **チェ・スンジュン** 1981年に撮られた写真です。生後10か月の子です。全部で4枚あるんですが、今はまず3枚を見せています。子ども、いや子どもですらない、乳児です。乳児が雑誌の中の時計を指さして、先生がたまたまアナログ時計をしていたので、こうやって見せてあげる。すると乳児に音を聞かせるんです。

乳児は話せないけれど、表情でコミュニケーションして、好奇心の状況が生まれました。体験が起きたんです。次に何が起きるかというと、雑誌の紙にある時計の画像からも音が出るのか気になって、生後10か月だから、胎生期から数えれば20か月くらいの存在ですよね。話せなくても、言語を発する前から仮説を立てて実験することはできるんです。

そしてその瞬間を記録として残すことで、その強いイメージ、その強い意味を語る場所が、イタリアの小都市の教育アプローチです。実際にはもう小都市ではなく、今は世界中からその教育を見に来ます。そこで有名な「子どもの100の言語」の話をしていて、僕たちもそこに影響を受けて似た話をしています。

## プログラムではなくプロジェクト: 予測不能な学びのプロセス    *10:39*

<span class="paragraph-timestamp" data-ts="10:39">10:39</span> **チェ・スンジュン** 僕たちはどんな教育をしているかと聞かれたら、programmazioneの反対だと言います。programmazioneはイタリア語で「program」です。つまりプログラムの反対を実践する教育です。普通どこかで教育をしていると言うと、「どんなプログラムですか？」「プログラムは？」って聞かれますよね。固定されたもの、計画されたものを遂行するニュアンスが「program」には強い。でもプログラムでないというのは、教育に関する初期仮説が実際に教えてみると現実と違って、進行しながら変わるということです。相手が人間だからです。

だから学びや勉強は、手続きが予測可能に段階的に進む、プログラムのような一次元の問題ではありません。知識形成は集団が経験するプロセスです。そして子どもは、先ほどの写真のように、自分なりの理論を作ります。先に答えを与えて、その理論を作る機会を奪うことは、実は奪ってしまっているんです。

過去の人類史をなぞることになるかもしれないけれど、自分で気づいて試せたかもしれない。先に答えを与えるのはネタバレです。気づけることを過度に先回りして教えない、気づけるように支援する、そういう哲学を持つ教育です。

## 教師の役割: 迷宮のアリアドネの糸    *12:10*

<span class="paragraph-timestamp" data-ts="12:10">12:10</span> **チェ・スンジュン** なので僕たちもそこに刺激を受けて、僕たちなりに図式化してみました。不確実な状況で子どもと教師が対話し、探究し、試行錯誤して、「あ、これ遊びが何かになるかも」と考える。これを僕たちは「遊びの種」と呼んでいます。

こうした健全な期待がある状況でも一直線ではありません。この図では曲線で描いています。一気にまっすぐ進まないという意味です。この時点で僕たちの教師は「考える時間を守ろう」という表現を使います。考える時間、熟考する時間。これを教えて次、次、と急いで何かを作るのではなく、むしろうろうろする感覚です。

でもそこに点があります。それは子どもの話を聞く時間かもしれないし、教師が記録を共有してフィードバックを交換する時間かもしれないし、保護者と共有する場かもしれない。でもその瞬間はとても重要なポイントです。潜在的な方向を探って新しい方向へ進む機会だからです。

だからRLエピソードを収録したときに、キム・ソンヒョンさんが高エントロピートークンの話をしていましたよね。複数の経路があり得る不確実な状況では、それがとても決定的になり得る。そこからどの道を選ぶかで推論経路がずっと良くなる可能性がある。その瞬間に何をするかで生成の質が大きく変わる。latent stateを得るわけです。

だからとても似たメンタルイメージだと思います。こうやって展開し続けて前に進める。別の支流でまた合流することもあるし、同時並行で進むこともある。教育分野では最近これをよくスパゲッティにたとえます。Reggio Emiliaだけでなく、始まりも終わりもなくスパゲッティのようにもつれて、新しい方向へ常に分岐する感覚が教育学でよく語られます。

学びは意味を構築するプロセスです。ギリシャ・ローマ神話にテセウスの迷宮がありますよね。ミノタウロスの迷宮。でもテセウスが迷宮を抜けられた理由は、アリアドネという王女が糸を渡したからです。糸を渡せば迷宮の中で通ってきた道の記録が残る。それで迷宮を抜けられる。

だから教師がその役割を担えると彼らは考えています。教師は糸玉を持つ人で、関係のネットワークを編み、相互作用とコミュニケーションを重要な経験にしていく。教師のコンピテンシーと役割を強調する部分があります。さっきの迷宮にもつれるという比喩ですね。

## 学びの足跡をたどるうえでのドキュメンテーションの重要性    *15:14*

<span class="paragraph-timestamp" data-ts="15:14">15:14</span> **チェ・スンジュン** 勇気ある航海のためには、足跡を残すことがある意味重要です。方向がわかるし、どこが安全だったかもわかるし、どんな実験や試みをしたかもわかる。つまりデータを蓄積することが重要で、それを教育で強調するのが、先ほどのReggio Emiliaの試みです。

なので僕たちもそこに影響を受けて、1997年からドキュメンテーションを重要視してきました。だからこそ「未来はすでにここにある。ただ均等に分配されていないだけだ」と、ある程度自信をもって言えるんです。これから僕たちの事例を紹介するからです。これは僕自身が参加したものです。

1997年に僕たちが試したこと、イタリアから学んだ時点から、記録を全部デジタル化しました。ざっと見せると、これが最初の記録です。始まりは1984年ですが、この形式で記録し始めたのは1997年です。子どもたちと交わした話、子どもたちが描いた絵、試したことの試行錯誤、そういったものを初めて記録として残しました。でもこれはまだ「読める記録」にとどまっていました。

<span class="paragraph-timestamp" data-ts="17:15">17:15</span> **チェ・スンジュン** それで2010年代に入り、2008年、つまり2010年代前にGoogle Docsのようなものを導入して組織内共有を始めました。さらに積極的にやるために、2013年ごろKanbanに刺激を受けて、Joel Spolskyでしたっけ、彼が作ったTrelloというツールがありましたよね。記録をカードとして動かすようなツールを、僕たちは2013年から遊び、あるいは遊びを通じた学びの記録に使ってきました。

ここで強調したいのは、こういう不確実な状況で試すとき、記録を残すことは役立つということです。でも想像できるとおり、これは手間がかかる。記録するには写真を撮って整理して、それについて考えを書き残す時間が必要です。いつもあるジレンマは、「こうするのが良い」とわかっていても、どう簡単にするかでした。さっき話した段階も、まだ教師が記録しなければいけない。でも記録しながら考えるので良い面は確実にありますが、かなり労力集約的でもあります。そこだけ指摘しておきます。

なので、子どもと教師が一緒に仮説を立てて、それを検証するリビングラボを目指す文化がありました。でも問題は、これがスケーリングで難しさを持つことです。
